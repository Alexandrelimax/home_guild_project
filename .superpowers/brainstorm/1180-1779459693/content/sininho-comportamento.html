O que acontece quando o player clica no sininho?

O sininho aparece no header com um badge numérico. Ao clicar, qual é o comportamento?

A

Dropdown inline com lista de notificações

Um painel abre abaixo do sininho, sem sair da página atual. Cada notificação tem um link "Ver evento →" que navega para /events. Fechar o painel marca como lido (badge some).

🔔2
NOTIFICAÇÕES
⚡ Novo Evento!
Lavar o Banheiro
Ver evento →
✓ Quest Aprovada!
Limpar a Cozinha
Ver conquista →
B

Clique no sininho navega direto para /events

Simples e direto. O sininho é apenas um atalho de navegação — clicar vai para a página de Eventos onde o player vê o card. O badge numérico some ao visitar a página.

🔔1
↓ navega direto para /events
⚡ EVENTO ESPECIAL · PENDENTE
Lavar o Banheiro
Complete esta missão para ganhar o badge exclusivo.
C

Dropdown com toast de contexto + link

O sininho abre um dropdown compacto mas com uma mini-prévia do badge que o player vai ganhar — cria o gancho visual do prêmio antes de navegar. Um botão "Ir para Eventos" leva à página.
